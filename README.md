# Is Subset Possible?

In this problem we need to enter an array and an integer and then return true if subset of entered integer is possible by integers present 
in array.

# Example 
-----------------------------
Enter size of array

5

Enter array of integers

2 5 11 19 20

Enter number

13


Subset of '13' possible?

 ---->  true
------------------------------

Enter size of array

2

Enter array of integers

2 3

Enter number

13


Subset of '13' possible?

 ---->  true
